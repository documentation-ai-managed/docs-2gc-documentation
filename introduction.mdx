## Key Features

2GC Documentation offers a range of features to enhance your documentation workflow. Our platform supports multiple content formats, including Markdown and rich text, and provides advanced search capabilities to help users find the information they need quickly.

<Tabs>
  <Tab title="Markdown Support" icon="file-text">
    Write documentation using familiar Markdown syntax with extensions for tables, code blocks, and more.
    
    <CodeGroup tabs="Basic,Advanced">
      ```markdown
      # Document Title
      
      This is a simple document.
      ```
      ```markdown
      # Advanced Document
      
      | Feature | Description |
      |---------|-------------|
      | Search | Fast indexed search |
      | Export | Multiple format exports |
      ```
    </CodeGroup>
  </Tab>
  <Tab title="API Access" icon="terminal">
    Access your documentation programmatically through our REST API.
    
    <Request tabs="cURL,JavaScript" show-lines="true">
      ```bash
      curl -X GET https://api.2gc.com/docs
      ```
      ```javascript
      fetch('/api/docs')
        .then(response => response.json())
        .then(data => console.log(data));
      ```
    </Request>
  </Tab>
</Tabs>

<Callout kind="info">For the best experience, we recommend using the latest version of your preferred browser and ensuring JavaScript is enabled.</Callout>
